# Design Tokens

## Border Radius

| Token         | Utility      | Value     | Use case                            |
| ------------- | ------------ | --------- | ----------------------------------- |
| `--radius-md` | `rounded-md` | `0.5rem`  | Content: cards, dropdowns, modals   |
| `--radius-sm` | `rounded-sm` | `0.25rem` | Components: buttons, inputs, badges |
